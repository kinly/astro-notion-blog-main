游戏类型分类与术语解析

2025-03-16

一、核心游戏类型

1. MMORPG（大型多人在线角色扮演游戏）
  • 特点：多人在线、持续开放世界、角色扮演元素
  • 代表作品：
    • 《魔兽世界》《剑灵》《剑侠情缘网络版 3》
    • 《上古世纪》《永恒之塔》《怪物猎人 Online》
2. RPG（角色扮演游戏）
  • 子类：
    • ARPG（动作角色扮演游戏）：强调即时战斗操作（如《暗黑破坏神》系列）
    • SLG（策略角色扮演游戏）：结合策略与角色扮演（如《火焰之纹章》系列）
3. RTS（即时战略游戏）
  • 经典作品：
    1. 《星际争霸》系列：暴雪娱乐开发，多种族深度战略对战。
    2. 《红色警戒》系列：Westwood Studios 开发，历史题材与科幻元素结合。
    3. 《帝国时代》系列：历史文明发展与战争模拟。
    4. 《魔兽争霸》系列：奇幻背景的 RTS 代表作。
    5. 《全面战争》系列：即时战略与回合制结合的史诗级战争游戏。
4. 射击游戏
  • 子类：
    • FPS（第一人称射击）：《使命召唤》《战地》《反恐精英》
    • TPS（第三人称射击）：《GTA》系列《战争机器》

二、特殊游戏类型

1. Roguelike
  • 核心特征：
    • 随机生成地牢、回合制、永久死亡、磁贴式图像
    • 代表作品：《NetHack》《以撒的结合》《哈迪斯》
2. DBG（卡牌构筑游戏）
  • 代表作品：
    • 《杀戮尖塔》（PC 端）
    • 《月圆之夜》《枪与香蕉》（移动端）
    • 《奇迹时代》（卡牌构筑鼻祖）
