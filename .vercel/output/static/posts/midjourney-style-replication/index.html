Midjourney 复刻任何你想要的风格

2025-03-16

在使用 Midjourney 时会遇到一个常见的挑战：看到一个引人入胜的图像，想要在自己的作品中重现这种风格，但不知道如何在 Midjourney 中实现这一点。这时，Style Reference(--sref) 功能就派上用场了。

一、没有种子的情况

有了 --sref 万物皆可风格化：

简单来说，在命令后面加上—sref 123456，这串数字，再用--sw 控制风格

--sref（风格参考）功能允许用户轻松地将首选图片的风格应用到新的创作中。上传一张图片将 [image_url] 内容替换 

/imagine prompt --sref [image_url]

多重风格参考：可以使用空格分隔多个图像 URL，例如： --sref urlA urlB urlC

调整风格参考强度：使用 --sw（风格权重）参数调整风格强度，范围从 0 到 1000，默认值为 100。

调整不同图片的权重：在图片 URL 后添加双冒号和一个值以调整权重，例如： 

--sref urlA::2 urlB::3 urlC::5.

高级技巧

调整样式权重 (--sw)

–sw（样式权重）参数决定了样式参考对输出的影响程度。该参数范围从 0 到 1000，允许精细调整样式在结果中反映的程度。

默认值为 100 时，增加样式权重会增强样式参考对输出的影响。

使用 sref 代码

使用 sref 代码非常简单。只需在 Midjourney 的提示结尾处添加相应的代码。例如，

/imagine prompt [your description] --sref 5000

这可能会应用特定的艺术风格。例如，向提示中添加一个 sref 代码可以在新图像中复制预设风格。

结合多个样式引用

可以将多张图片作为样式参考以融合不同的美学风格。通过为每张图片分配权重，您可以控制它们的影响并突出特定的风格。例如，使用 --sref 151032020 和 --sref 251 代码一起将生成具有铝质纹理的动漫风格插图。

如何创建 sref 代码（Midjourney）？

有两条主要途径可以获得 Midjourney 的 sref 代码：

  1. 使用 --sref random --sref 随机命令：这是 Midjourney 的官方功能。当你在提示中添加 --sref random 时，Midjourney 将随机生成一个 sref 代码并应用相应的风格。这种方法的优点在于其简单直接，允许你探索各种风格。
  2. 使用 sref 代码库或共享平台：随着 sref 功能的流行，许多网站和平台已经出现，收集并分享高质量的 sref 代码。精选的 sref 代码库。浏览这些代码，查看示例图片和提示，并选择喜欢的风格。
